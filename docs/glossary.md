# KOWRAP Glossary

Last updated: 2026-05-21

## Core Terms

**줄바꿈 가능 지점 / break opportunity**

렌더러가 줄 끝에서 텍스트를 다음 줄로 넘길 수 있는 후보 위치. 모든 후보가 좋은 후보는 아니다.

**안전 줄바꿈 / safe break**

의미, 형태, 가독성, 레이아웃 안정성을 크게 해치지 않는 줄바꿈 위치. KOWRAP의 주요 예측 대상이다.

**나쁜 줄바꿈 / bad break**

긴 복합어, 전문용어, 고정 표현, 숫자·단위 표현, 기관명 등을 어색하게 끊어 독해 비용을 높이는 줄바꿈.

**강제 줄바꿈 / forced break**

사용자가 `<br>`, HWP 문단/줄 나눔, 수동 개행 등으로 직접 삽입한 줄바꿈. 자동 조판 실패를 사람이 보정한 흔적으로 볼 수 있다.

**복합명사 / compound noun**

여러 명사 또는 어근이 결합해 하나의 긴 단위를 이루는 표현. 예: `국가연구개발사업성과평가`, `공공데이터활용지원체계`.

**어절 / eojeol**

한국어 띄어쓰기 단위. 공백 기준 줄바꿈은 보통 어절 경계를 사용하지만, 긴 복합어 문제는 어절 내부에서 발생한다.

**음절 / syllable**

완성형 한글 한 글자에 가까운 단위. 음절 단위 줄바꿈은 구현은 쉽지만 의미를 해칠 수 있다.

**자소 / jamo**

한글 음절을 이루는 초성, 중성, 종성 단위. KOWRAP의 초기 목표는 자소 단위 조판이 아니라 음절/형태/통계 기반 줄바꿈이다.

## Layout Terms

**자간 / inter-character spacing**

글자 사이 간격. HWP 계열 문서에서 줄 맞춤을 위해 자주 수동 조절된다.

**장평 / glyph width scale**

글자 폭의 가로 비율 조정. 같은 텍스트라도 장평이 달라지면 줄바꿈 결과가 달라질 수 있다.

**양끝 정렬 / justification**

줄의 양쪽 끝을 맞추기 위해 공백, 자간, 글리프 폭 등을 조절하는 방식.

**줄바꿈 안정성 / line-break stability**

폰트, 렌더러, 폭, 장평, 자간이 조금 바뀌어도 줄바꿈 결과가 크게 흔들리지 않는 성질.

**렌더러 / renderer**

텍스트와 스타일을 실제 화면 또는 PDF 이미지로 배치하는 엔진. 예: 한컴오피스, 한컴독스 웹, 브라우저, PDF 렌더러.

**폰트 메트릭 / font metrics**

글자의 폭, 높이, 기준선 등 렌더링 계산에 쓰이는 수치. 폰트가 바뀌면 같은 문장도 다른 줄에서 끊긴다.

## Model Terms

**규칙 기반 모델 / rule-based model**

사전, 금칙, 정규화 규칙만으로 줄바꿈 후보를 계산하는 모델.

**통계 기반 모델 / statistical model**

빈도, PMI, branching entropy 등 말뭉치 통계로 결합 강도와 분절 가능성을 계산하는 모델.

**경량 모델 / lightweight model**

브라우저나 로컬 CLI에서 빠르게 실행되도록 작은 크기로 배포되는 ML 모델.

**도메인 오버라이드 사전 / domain override dictionary**

기관명, 사업명, 전문용어처럼 일반 모델이 잘못 끊기 쉬운 표현을 보호하거나 특정 위치에서만 끊도록 지정하는 사전.

**모델 카드 / model card**

모델의 학습 데이터, 도메인, 평가 결과, 실패 사례, 배포 범위를 기록하는 문서.

## Project Terms

**증거팩 / evidence pack**

문제 사례를 텍스트, 렌더링 조건, 이미지, 기대 동작으로 묶은 재현 가능한 자료 묶음.

**참조 구현 / reference implementation**

논문과 표준화 논의에서 동작을 확인할 수 있는 최소 구현체. 완성 제품이 아니라 합의 가능한 기준점이다.

**표준화 테스트 케이스 / standardization test case**

특정 구현체가 아니라 요구사항 자체를 검증하기 위해 만든 작은 예제.
